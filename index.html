Mon Mar 10, 2014 by 尹向旭

在personal-web的命令提示符中依次输入：

  • hugo server –watch –theme=personal-web
  • myblog »hugo server -t personal-web –buildDrafts
  • myblog »hugo –theme=personal-web –baseUrl="https://yinxx.github.io” –buildDrafts

public根目录下依次输入：

  • git init 将public文件夹转换为仓库

  • git add . 选择所有的内容

  • git commit -m “first commit” 提交至个人站点中

  • 第一条：git init

  • 第二条：git add . （切记：add后面加一个空格再加一个“.”）

  • 第三条：git commit -m “我的hugo博客”

  • 将本地public内容与站点进行关联,后面的地址是:GitHub官网+你的账户名+你新建的站点名+.git

  • 第四条：git remote add origin https://github.com/yinxx/yinxx.github.io.git

  • 将public刷新到站点中 地址与上面相同.第一次会叫你输入你GitHub的账户密码.密码是暗文照常输入就行

  • 第五条：git push -u origin master
